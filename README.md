# Spider IA 🕷️

Un chatbot interattivo con ragnatela SVG animata, costruito in React + Vite.

## Come funziona

Scrivi un messaggio nella casella di testo: la risposta appare su uno dei nodi della ragnatela, che vibra e si illumina ad ogni interazione.

## Avvio

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
